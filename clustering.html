Neste capítulo iremos abordar a tarefa de Clusterização com a biblioteca scikit-learn. Para tanto, iremos implementar um programa que irá agrupar em clusters dados referentes aos valores de altura de mães e filhas. O algoritmo de Clusterização que iremos utilizar será o K-Means. Este tutorial será realizado em 7 etapas:

  • Etapa 1 - O Algoritmo: Apresentação do algoritmo que será implementado na aula tutorial.
  • Etapa 2 - O Dataset: Definição do dataset que será utilizado
  • Etapa 3 - Implementação do K-Means para k = 2: Implementação da Clusterização com o algoritmo K-Means para gerar dois clusters.
  • Etapa 4 - Implementação do K-Means para k = 3: Implementação da Clusterização com o algoritmo K-Means para gerar três clusters.
  • Etapa 5 - Apresentação de Resultados - Gráfico de Dispersão: Visualização dos resultados da clusterização com Gráfico de Dispersão
  • Etapa 6 - Apresentação de Resultados - Diagrama de Voronoi: Visualização dos resultados da clusterização com Diagrama de Voronoi.

4.1 Etapa 1 - O Algoritmo

Nesta etapa iremos descrever, de forma geral, o algoritmo que será implementado ao longo deste tutorial. O fluxograma da Figura 1 indica as principais atividades que serão implementadas.

1. Definir o dataset:

Dataset é o nome atribuído ao conjunto de dados ou documentos utilizado a ser utilizado na tarefa de Machine Learning. O dataset pode ser formado por valores numéricos contínuos ou discretos, ou ainda por textos, imagens, áudios, vídeos, etc.

A construção do dataset envolve decidir quais dados serão utilizados. Ele poderão ser privados (particulares de empresas, instituições, etc) ou públicos (dados recuperados da Web por meio de downloads, web scraping, etc). Em nosso caso, iremos utilizar um dataset disponibilizado no livro Estatística Elementar, de Mario Triola, com dados referentes ao conjunto de alturas de uma família.

2. Definir K:

O algoritmo K-Means não consegue descobrir automaticamente a melhor quantidade de clusters para um dataset. Dessa forma, devemos especificar para o algoritmo a quantidade de clusters que desejamos gerar. Essa quantidade é representada pelo valor de K. Assim, para K = 2, dois clusters serão gerados pelo algoritmo. Para K = 3, três clusters serão gerados, etc. Nesta etapa iremos definir o valor de K para a nossa clusterização dos dados do livro de Triola.

3. Clustering dos Dados:

Nesta etapa o agrupamento via clustering será realizada pelo algoritmo K-Means para os dados definidos no dataset.

4. Apresentar Resultados:

Os resultados estão relacionados com a visualização dos clusters gerados de forma gráfica. Nesta etapa, iremos apresentar os clusters formados utilizando Gráfico de Dispersão e Diagrama de Voronoi.
